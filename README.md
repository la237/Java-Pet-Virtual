# 🐾 Simulador de Animal de Estimação Virtual

## 📖 Sobre o Projeto

Este projeto consiste em um simulador de animal de estimação virtual desenvolvido para praticar os conceitos de **Programação Orientada a Objetos (POO)**. Nele, o usuário pode interagir com um animal virtual realizando ações como alimentar, brincar e fazê-lo descansar.

## 🎯 Objetivo

Aplicar conceitos fundamentais de POO, como:

- Classes e Objetos
- Atributos e Métodos
- Encapsulamento
- Instanciação de Objetos

## ⚙️ Funcionalidades

- Alimentar o animal
- Brincar com o animal
- Fazer o animal dormir
- Visualizar seu status

## 🚀 Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)
